# 🔗 ReFi Token Flow - Blockchain Transparency

Aplicación interactiva para visualizar el flujo de tokens FCEA en blockchain, demostrando la transparencia de las finanzas regenerativas (ReFi).

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io)

## ✨ Características

- 🕸️ **Red de nodos interactiva** - Visualiza conexiones entre wallets
- 📊 **Diagrama Sankey** - Flujo de tokens desde mint hasta destinatarios
- 💰 **Balance de wallets** - Estado actual de cada participante
- 📋 **Registro de transacciones** - Historial completo con hashes

## 🚀 Despliegue en Streamlit Cloud

1. Haz fork de este repositorio
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta tu cuenta de GitHub
4. Selecciona este repo y `app.py`
5. ¡Deploy!

## 🏃 Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
